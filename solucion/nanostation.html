Mi NanoStation no enlaza – Soluciones paso a paso

Inicio Sobre Blog Artículos Guías Técnicas Proyectos Solución y Herramientas Sígueme

Los equipos NanoStation de Ubiquiti son muy utilizados en enlaces inalámbricos punto a punto para conectar casas, negocios, cámaras de seguridad y redes remotas. Sin embargo, uno de los problemas más comunes es que las antenas no logran enlazar o conectar entre sí. En esta guía te mostramos cómo diagnosticar este problema paso a paso.

1. Verificar la frecuencia

Uno de los errores más frecuentes es que ambos equipos estén configurados en diferentes bandas de frecuencia.

  • 2.4 GHz (NanoStation M2)
  • 5 GHz (NanoStation M5)

Ambos dispositivos deben trabajar en la misma banda. También se recomienda evitar frecuencias saturadas.

Consejo profesional: Utiliza el escaneo de espectro en el sistema airOS para elegir una frecuencia con menor interferencia.

2. Revisar el canal inalámbrico

Aunque ambos equipos estén en la misma banda, el canal también debe coincidir.

Ejemplo correcto:

  • Antena emisora: 5745 MHz
  • Antena receptora: 5745 MHz

También revisa el ancho de canal (20 MHz, 40 MHz, etc.).

3. Seguridad inalámbrica

Si el tipo de seguridad o contraseña no coinciden, el enlace no se completará.

  • Usa WPA2-AES
  • Contraseña exactamente igual
  • Respeta mayúsculas y minúsculas

Muchos técnicos pierden horas por este detalle.

4. Alineación de las antenas

Una mala alineación provoca señal baja, enlace inestable o fallas de conexión.

  1. Configura una antena como Access Point.
  2. La otra como Station.
  3. Observa el nivel de señal RSSI.
  4. Ajusta lentamente hasta lograr la mejor señal.

Nivel recomendado: entre -40 y -60 dBm.

5. Nivel de señal

Si la señal es muy baja, el enlace puede no establecerse o conectarse y desconectarse.

Revisa:

  • Obstáculos como árboles o edificios
  • Distancia excesiva
  • Interferencias

Si el RSSI es menor a -75 dBm, el enlace será inestable.

6. Firmware actualizado

Actualizar el firmware puede resolver problemas de compatibilidad.

  • Usa la misma versión en ambos equipos
  • Reinicia después de actualizar
  • Descarga firmware oficial

Otros puntos importantes

  • Región configurada correctamente
  • Modo AirMAX activado
  • Modo Bridge habilitado
  • Direcciones IP en la misma red
  • Cableado en buen estado

Consejos de experto

Como especialista en redes y enlaces inalámbricos, te recomendamos:

  • Realizar pruebas antes de la instalación definitiva
  • Evitar interferencias de otras redes
  • Usar torres o mástiles para mejor línea de vista
  • No usar anchos de canal grandes en zonas saturadas
  • Documentar cada enlace

¿Necesitas ayuda profesional?

Si después de estas pruebas el problema continúa, puede ser necesario realizar un análisis profesional del enlace. Puedes contactarnos para diagnóstico, diseño o instalación de enlaces inalámbricos.
