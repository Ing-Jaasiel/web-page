Normas Clave TIA/EIA para Cableado Estructurado

TIA/EIA-568 (Series: 568-C, y sus partes)

Título General:

Telecommunications Cabling Standards (Estándares de Cableado de Telecomunicaciones).

Impacto:

  • Topología: Define la topología en estrella como la arquitectura recomendada.
  • Subsistemas: Divide el sistema de cableado en seis subsistemas principales (Entrada de Servicios, Sala de Equipos, Cableado Backbone, Armario de Telecomunicaciones, Cableado Horizontal, Área de Trabajo).
  • Especificaciones de Componentes: Define las características de rendimiento mínimas para cables, conectores, patch panels, etc.
  • Longitudes Máximas: Establece las longitudes máximas permitidas para los cables.
  • Esquemas de Terminación: Define los esquemas de terminación para los conectores RJ45 (T568A y T568B).

TIA-569 (Series: 569-D)

Título General:

Telecommunications Pathways and Spaces (Vías y Espacios de Telecomunicaciones).

Impacto:

  • Diseño de Espacios: Especifica los requisitos para los armarios de telecomunicaciones y las salas de equipos.
  • Vías de Cableado: Define las recomendaciones para las vías de cableado (canalizaciones, bandejas, conductos).

TIA-606 (Series: 606-C)

Título General:

Administration Standard for Telecommunications Infrastructure (Estándar de Administración para la Infraestructura de Telecomunicaciones).

Impacto:

  • Identificación y Documentación: Establece un sistema uniforme para la identificación de componentes.
  • Etiquetado: Define los requisitos para el etiquetado de los componentes del cableado.

TIA-942 (Series: 942-B)

Título General:

Telecommunications Infrastructure Standard for Data Centers (Estándar de Infraestructura de Telecomunicaciones para Centros de Datos).

Impacto:

  • Diseño Específico para Centros de Datos: Proporciona directrices adicionales para entornos de centros de datos.
  • Topologías y Distancias: Recomienda topologías específicas y define las distancias máximas en centros de datos.

Otros Estándares Relevantes

  • TIA-862 (Series: 862-B): Building Automation Systems Cabling Standard (Estándar de Cableado para Sistemas de Automatización de Edificios).
  • TIA-TSB (Telecommunications Systems Bulletins): Boletines técnicos con aclaraciones y actualizaciones.

Impacto General en el Diseño e Instalación

El cumplimiento de las normas TIA/EIA asegura la interoperabilidad, el rendimiento, la fiabilidad, la escalabilidad y facilita el mantenimiento de los sistemas de cableado estructurado. También suele ser un requisito para las garantías de los fabricantes.
