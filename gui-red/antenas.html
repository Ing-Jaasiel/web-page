Configuración de Enlace Inalámbrico: Modo Punto de Acceso (AP)

Inicio Artículos Guías Proyectos S&H Asesoría

Un enlace punto a punto (PtP) permite conectar dos ubicaciones físicas mediante antenas inalámbricas en 5GHz sin necesidad de cableado estructurado o fibra óptica.

¿Para qué sirve un enlace PtP?

  • Conectar oficinas o sucursales
  • Llevar Internet a sitios remotos
  • Interconectar cámaras de seguridad
  • Crear enlaces entre edificios sin cable

Material Necesario

  • 2 radios inalámbricos (NanoStation Loco M2, M2 o Arimax 5)
  • Adaptadores PoE
  • Cables Ethernet Cat5e o Cat6
  • Computadora para configuración
  • Soporte o poste para montaje

Paso 1: Instalación Física

Monta ambas antenas en puntos elevados con línea de vista directa. Evita obstáculos como árboles o estructuras metálicas.

Paso 2: Acceso al Equipo

  1. Configura tu PC con IP estática en la red 192.168.1.x
  2. Ingresa en el navegador a: http://192.168.1.20

Diferencias Según el Firmware (Muy Importante)

Dependiendo de la versión de firmware instalada en la NanoStation Loco M5, el proceso de acceso inicial puede variar.

Firmware Antiguo (airOS clásico)

  • Usuario: ubnt
  • Contraseña: ubnt
  • No solicita región al ingresar

Después de ingresar se recomienda cambiar inmediatamente la contraseña por seguridad.

Firmware Nuevo (Asistente Inicial)

  • Solicita seleccionar país o región
  • Permite elegir idioma
  • Obliga a crear nuevo usuario y contraseña
  • Se deben aceptar los términos de uso

En este caso no se utiliza ubnt/ubnt, ya que el sistema requiere nuevas credenciales por seguridad.

⚙️ Configuración del Equipo Emisor (Access Point)

El equipo emisor será el que genere la señal inalámbrica que el otro dispositivo recibirá.

Configuración de Red

  • Modo de red: Bridge
  • Asignar IP estática (ejemplo 192.168.1.30)
  • Máscara: 255.255.255.0
  • Gateway según tu red
  • DNS recomendado: 8.8.8.8

Configuración Inalámbrica

  • Wireless Mode: Access Point (AP)
  • Crear un SSID personalizado
  • Seguridad: WPA2-AES
  • Seleccionar canal fijo
  • Ancho de canal: 20 o 40 MHz

📡 Configuración del Equipo Receptor (Station)

El equipo receptor será el encargado de conectarse al Access Point para establecer el enlace punto a punto.

  • Wireless Mode: Station o Client
  • Buscar y seleccionar el SSID configurado en el AP
  • Ingresar la misma contraseña WPA2
  • Configurar IP estática en la misma red
  • Modo de red: Bridge

Una vez aplicados los cambios, el enlace debería establecerse automáticamente si ambos dispositivos están correctamente alineados y en la misma frecuencia.

Alineación y Pruebas

Alinea ambas antenas hasta obtener buena intensidad de señal y realiza pruebas de ping entre extremos.

Recomendaciones de Seguridad

  • Cambiar credenciales por defecto
  • Actualizar firmware
  • Usar contraseñas robustas
  • Seleccionar canales poco saturados

Conclusión

Configurar un enlace punto a punto con NanoStation Loco M5, Loco M2, Arimax 5, LiteBeam y PowerBeam es una solución eficiente para extender redes inalámbricas en 2.4GHz y 5GHz de forma estable y profesional.
