Configuración de Enlace Inalámbrico: Modo Punto de Acceso (AP)

Guía técnica profesional para la configuración de equipos Ubiquiti Airmax M5.

Introducción Técnica

El primer paso para un enlace exitoso es configurar la antena que estará conectada directamente al router principal (Internet). Esta antena funcionará como el "servidor" inalámbrico que enviará la señal al punto remoto.

1. Acceso y Credenciales de Fábrica

Conecta la antena a tu computadora y asegúrate de estar en el segmento de red 192.168.1.x. Accede vía navegador a la IP por defecto.

  • IP por defecto: 192.168.1.20
  • Credenciales: Usuario: ubnt / Password: ubnt
  • Selección de País: Es vital elegir el país correcto para cumplir con las regulaciones locales de potencia y frecuencias permitidas.

2. Configuración de Red (Network)

Antes de cambiar los parámetros inalámbricos, debemos asegurar la administración del equipo.

  • Network Mode: Bridge (Puente).
  • IP Address: Cambia la IP estática (ej. 192.168.1.25) para evitar conflictos si tienes otros equipos en la misma red.
  • Gateway: Coloca la dirección IP de tu router principal.

3. Parámetros Inalámbricos (Wireless)

Aquí es donde definimos el comportamiento de la antena como emisora de señal.

  • Wireless Mode: Access Point (Punto de Acceso).
  • WDS (Transparent Bridge Mode): ¡Activado! Esto es fundamental para que el tráfico de red pase de forma transparente.
  • SSID: Define el nombre de tu enlace (Ej: Enlace_Jaasiel_P1).
  • Channel Width: Se recomienda 20MHz para máxima estabilidad o 40MHz para mayor velocidad.

4. Seguridad y Cifrado

Nunca dejes un enlace abierto. La seguridad inalámbrica protege la integridad de los datos transmitidos.

  • Security: WPA2-AES.
  • WPA Preshared Key: Define una contraseña robusta que solo tú conozcas.

¡Eso es todo! Con un poco de práctica, crimparás conectores como un profesional. Recuerda, la clave está en el orden de los colores y en un buen crimpado. ¡Inténtalo!
